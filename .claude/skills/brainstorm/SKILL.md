# Brainstorm Skill

Turn vague ideas into actionable specifications.

## When This Skill Activates

- User describes idea without specifics
- Starting greenfield project
- Exploring feasibility

## Questions to Ask

1. **What problem does this solve?**
2. **Who is this for?**
3. **What's the simplest version that delivers value?**
4. **What already exists? (prior art)**
5. **What's the riskiest assumption?**

## Output

Structure the exploration into:
- Problem statement
- Target user
- MVP scope
- Open questions
- Recommended next step (/build, /research, or break down further)

## Exit Criteria

- [ ] Problem clearly stated
- [ ] User identified
- [ ] MVP scope defined
- [ ] Ready for /build or identified blockers
